## 🌟 Overview

**StressAI** is a sophisticated wellness tool that leverages the power of computer vision and behavioral analytics to quantify mental well-being. Unlike traditional surveys, StressAI analyzes real-time interaction data and facial markers to provide an objective stress profile.

## 🚀 Key Features

- 🖱️ **Behavioral Fingerprinting**: Analyzes mouse jitter, typing cadence, and interaction velocity.
- 👁️ **Facial Bio-markers**: Detects eye fatigue, micro-expressions, and skin tone changes using TensorFlow.js.
- 📊 **Dynamic Visualization**: Interactive Recharts dashboards showing your stress "Radar".
- 🧘 **Actionable Insights**: Instant recovery protocols (breathing exercises, physical breaks) tailored to your data.
- ⚡ **High Performance**: Built with Vite 6 and React 19 for sub-second analysis.

## 🧠 The Science Behind It

StressAI operates on two primary data streams:

1.  **Cognitive Load (Behavioral)**: Based on the *Fitts' Law* and typing variability studies, we measure the precision of movements which degrades under high stress.
2.  **Physical Indicators (Facial)**: We track the *Orbicularis Oculi* muscle activity and skin reflectance which correlates with autonomic nervous system arousal.
